# My-Application
Задание 1
Какой стек лучше подходит для учебной Android-разработки и почему?
Ответ: я думаю, что для учебной Android-разработки лучше подходит Android SDK + Jetpack Compose (Kotlin), так как
1)	Kotlin – понятный язык для начинающих, официальный язык Android от  Google
2)	Jetpack Compose – удобный и наглядный фреймворк, пооностью родной для Android, легко понять, как данные связаны с экраном
3)	Google активно развивает Compose, существует много примеров использования
4)	Для использования не нужно изучать лишние технологии

3)	Определила, какие элементы проекта относятся к Android SDK, какие — к Jetpack, а какие — к инструментам сборки:
- AndroidManifest.xml – Android SDK 
- MainActivity.kt - Android SDK + Jetpack
- папка /res (ресурсы) – Android SDK
- Gradle-файлы – инструменты сборки

4) Описание назначения:
- AndroidManifest.xml – сообщает системе о приложении, перечисляет все экраны, указывает название, версию, иконку 
- MainActivity.kt – главная точка входа в приложение, управляет жизненным циклом экрана и содержит UI
- Gradle-файлы – указывают библиотеки, которые нужны проекту и управляют сборкой приложения
